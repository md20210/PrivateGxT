# PrivateGxT - RAG-powered Document Chat

**PrivateGxT** is a Retrieval-Augmented Generation (RAG) showcase that enables intelligent chat interactions with your documents. Upload PDFs, DOCX, or TXT files, and ask questions powered by multiple LLM providers.

## Features

- **Multi-Format Support**: PDF, DOCX, TXT document upload
- **RAG Technology**: ChromaDB vector database for intelligent retrieval
- **Multi-LLM Gateway**: Anthropic Claude, Grok, Ollama (local models)
- **Multilingual**: German, English, Spanish interface
- **Source Attribution**: See which document chunks were used for answers
- **Real-time Stats**: Track documents, chunks, and messages
- **Responsive Design**: Works on desktop, tablet, and mobile

## Quick Start

```bash
# Install dependencies
npm install

# Start dev server
npm run dev

# Build for production
npm run build
```
